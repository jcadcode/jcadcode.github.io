HTML Cheat Sheet

Document Template
Tags Description
<!DOCTYPE html> The declaration is not an HTML tag. It is an "information" to the browser about what document type to expect.
<html lang="en"></html> Used to identify the language of text content on the web. This information helps search engines return language specific results, and it is also used by screen readers that switch language profiles to provide the correct accent and pronunciation.
<head></head> Sets off the title & other info that isn't displayed Nested within the <html></html> tags
<body></body> Sets off the visible portion of the document
Head Template
Tags Description
<meta> Metadata is data (information) about data. The HTML5 specification encourages web developers to use the UTF-8 character set, which covers almost all of the characters and symbols in the world! Example <meta charset="utf-8">
<title></title> Puts name of the document in the title bar; when bookmarking pages, this is what is bookmarked
<link> used for defining a link to an external document like CSS and JavaScript. Example <link href="mystyle.css" rel="stylesheet">
Body Template
Tags Description
<header></header> used to define the header for a document or a section as it contains the information related to the title and heading of the related content.
<h1></h1> -- > <h6></h6> Creates headlines -- H1=largest, H6=smallest
<main></main> Main content, can only be used once per document.
<div></div> Used to format block content with CSS like id (#) and class (.).
<nav></nav> The nav element represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents.
<footer></footer> Typically contains information about the author of the section, copyright data or links to related documents.
